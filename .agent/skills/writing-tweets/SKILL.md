# Writing Viral Tweets

This skill enables the agent to help users craft high-performing tweets for x.com. It follows a structured approach of research, drafting, and optimization for engagement.

## When to use this skill
- When the user wants to write a tweet or X post about a specific topic.
- When the user needs a thread or a single post that is optimized for clicks and engagement.
- When the user asks for "clickbaity" or "CTA-friendly" social media content.

## Workflow
1. [ ] **Gather Topic**: If the user hasn't provided a specific topic, ask for one.
2. [ ] **Research**: Use `search_web` to find current trends, facts, or popular opinions related to the topic.
3. [ ] **Drafting**: Create 2-3 variations of the tweet based on the research.
   - **Concise**: Keep it under 280 characters.
   - **Clickbaity**: Use a hook that grabs attention.
   - **CTA**: Include a clear Call To Action.
4. [ ] **Refine**: Present the drafts to the user for feedback.
5. [ ] **Finalize**: Provide the final version of the tweet.

## Instructions
- **The Hook**: Start with a strong statement or a question that piqued interest.
- **The Body**: Use bullet points or short sentences for readability.
- **The CTA**: Encourage users to reply, retweet, or click a link.
- **Visuals**: Suggest what kind of image or GIF would pair well with the tweet.

## Research Guidelines
- Always use `search_web` to find the latest "spicy" takes or data points about the topic.
- Look for common pain points or "did you know" facts to drive engagement.

## Example Draft Structure
"**[Hook]**
...
[Value Point 1]
[Value Point 2]
...
**[CTA]**"
